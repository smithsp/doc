ADIABATIC_ELEC¶

Definition

Use adiabatic electrons.

Comments

  • DEFAULT = .false.

ALPHA_E¶

Definition

Multiplies ExB velocity shear for spectral shift model.

Comments

  • DEFAULT = 1.0

ALPHA_MACH¶

Definition

Multiplies parallel velocity for all species.

Comments

  • DEFAULT = 0.0

ALPHA_P¶

Definition

Multiplies parallel velocity shear for all species.

Comments

  • DEFAULT = 1.0

ALPHA_QUENCH¶

Definition

Choices

  • ALPHA_QUENCH = 1.0: use quench rule
  • ALPHA_QUENCH = 0.0: use new spectral shift model

Comments

  • DEFAULT = 0.0

ALPHA_SA¶

Definition

Normalized pressure gradient.

Comments

  • DEFAULT = 0.0

AS_1¶

Definition

Species array of densities {\({\frac {n_{s}}{n_{e}}}}\).

Comments

  • DEFAULT = 1.0 , 1.0

BETAE¶

Definition

{\(\beta _{e}\,\!\)} defined with respect to {;math;`B_{rm {unit}},!`}.

Comments

  • DEFAULT = 0.0

B_MODEL_SA¶

Definition

0/1 to exclude/include the B(theta) factor in k_per.

Comments

  • DEFAULT = 1

DEBYE¶

Definition

Debye length/gyroradius.

Comments

  • DEFAULT = 0.0

DEBYE_FACTOR¶

Definition

Multiplies the debye length.

Comments

  • DEFAULT = 1.0

Definition

Comments

  • DEFAULT = 0.0

{\({\frac {\partial R_{maj}}{\partial x}}}\).

Comments

  • DEFAULT = 0.0

DRMINDX_LOC¶

DZMAJDX_LOC¶

Definition

{\({\frac {\partial Z_{maj}}{\partial x}}}\).

Comments

  • DEFAULT = 0.0

ETG_FACTOR¶

Definition

Exponent for ETG saturation rule.

Comments

  • DEFAULT = 1.25

FILTER¶

Definition

Sets threshold for frequency/drift frequency to filter out non-driftwave instabilities.

Comments

  • DEFAULT = 2.0

FIND_WIDTH¶

Definition

Choices

  • FIND_WIDTH = .true. : find the width that maximizes the growth rate
  • FIND_WIDTH = .false. : use width

Comments

  • DEFAULT = .true.

FT_MODEL_SA¶

Definition

1 uses trapped fraction at the outboard midplane.

Comments

  • DEFAULT = 1

GCHAT¶

Definition

Multiplies the curvature drift irreducible terms.

Comments

  • DEFAULT = 1.0

GEOMETRY_FLAG¶

Definition

Geometry type.

Choices

  • GEOMETRY_FLAG = 0: \(s-\alpha\)
  • GEOMETRY_FLAG = 1: Miller
  • GEOMETRY_FLAG = 2: Fourier
  • GEOMETRY_FLAG = 3: ELITE

Comments

  • DEFAULT = 1

GHAT¶

Definition

Multiplies the curvature drift closure terms.

Comments

  • DEFAULT = 1.0

GRADB_FACTOR¶

Definition

Multiplies the gradB terms.

Comments

  • DEFAULT = 0.0

IBRANCH¶

Definition

Choices

  • IBRANCH = 0: find two most unstable modes one for each sign of frequency, electron drift direction (1), ion drift direction (2)
  • IBRANCH = -1: sort the unstable modes by growthrate in rank order

Comments

  • DEFAULT = -1

IFLUX¶

Definition

Compute quasilinear weights and mode amplitudes.

Comments

  • DEFAULT = .true.

KAPPA_LOC¶

Definition

Elongation of flux surface, {\(\kappa \,\!}\).

Comments

  • DEFAULT = 1.0

KX0_LOC¶

Definition

kx0/ky ballooning mode offset.

Comments

  • DEFAULT = 0.0

KY¶

Definition

{\(k_{\theta }\rho _{s,{\rm {unit}}}\,\!}\) for single-mode call to TGLF.

Comments

  • DEFAULT = 0.3

KYGRID_MODEL¶

Definition

Choices

  • KYGRID_MODEL = 0: user defined with NKY modes up to KY equal spaced
  • KYGRID_MODEL = 1: standard ky spectrum for transport model

Comments

  • DEFAULT = 1

LINSKER_FACTOR¶

Definition

  • DEFAULT = 0.0

  • DEFAULT = 4

Comments

  • DEFAULT = 2

NEW_EIKONAL¶

Definition

Choices

  • NEW_EIKONAL = .true. : compute the eikonal
  • NEW_EIKONAL = .false. : use the eikonal computed on the last call to TGLF made with tglf_new_eikonal_in = .true.

Comments

  • DEFAULT = .true.
