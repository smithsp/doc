The General Atomics GACODE Suite¶

GACODE refers to a collection of serial and HPC software developed by the Theory Group at General Atomics, and distributed through the GACODE source code repository at GitHub.

Installation¶

We maintain public installations of GACODE via AToM. However, you can also install and build the GACODE suite from sources.

Geometry¶

A description of the unified method used to describe flux-surface geometry.

Rotation and Field Orientation¶

Some notes about plasma rotation and magnetic field orientation.

Experimental profiles¶

GACODE software works with the input.profiles format.
